BOX MODEL

Defination

A box model is used in Css and HTML tags to define the layout of Web pages

Box Model Main Properties

  1. Width
    Specifies width of elements and elements content area.
  2. Padding
    Space around an elements content area inside any defined borders
  3. Margin
    This is the space outside an elements border
  4. Border
    Visible area enclosing the element it may be styled depending on someones choice
  5. Height
    Specifies the height of the element and elements content area
